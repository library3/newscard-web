# 問題意識練習｜時事卡工作坊

互動式單頁網站，供「114-2 專題寫作指導」課程使用。學生透過閱讀時事新聞，練習提出研究問題、建立問題意識與理論假設。

**線上網址：** https://library3.github.io/newscard-web/

## 使用方式

1. 學生從首頁四張卡片中任選一張點入
2. 閱讀新聞事件摘要與來源
3. 依提示至 Google Classroom 的連結簡報作答（研究問題 → 問題意識 → 假設 → 驗證方式）
4. 需要引導時點擊「給我一點提示」，展開核心矛盾與觀點對照

## 四張時事卡

| # | 標題 | 核心議題 |
|---|------|----------|
| 1 | 高齡駕駛的兩難困境 | 道路安全 vs. 移動正義 |
| 2 | 誰的路？ | 行人路權 vs. 商家生計 |
| 3 | 捷運夢與空蕩蕩的公車 | 硬體建設期待 vs. 使用習慣 |
| 4 | 最危險的保護 | 個人理性 vs. 集體後果 |

## 技術說明

純 HTML / CSS / JS，無外部框架，單一檔案 `index.html`。字型使用 Google Fonts（Noto Sans TC）。

若要修改卡片內容，編輯 `index.html` 內的 `CARDS` 陣列即可，改完後 `git push` 即自動更新 GitHub Pages。
